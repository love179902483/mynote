#### 变量，就是一个标签，有非数字开头的，字母、数字、下划线组成，它的内容可以是数值、字符串、列表、元祖、字典。

1. 列表[]
```py
c = [1,2,3,4,5] 
d = ["张三","李四"]  
e = [1,2,3,4,"5","6"]
```

+ 用列表名[索引号],索引列表中的元素
d[0]表示列表d中的第一个元素"张三"

+ 用列表名[起: 止]表示切片，从列表中切片出相应的元素，***前闭后开***
```py
c[0:2]  #[1,2]
c[:]    #[1,2,3,4,5]
```
  ***注***: 切片时候如果只有一个`:`则表示获取全部列表.

+ 用列表名[起:止:步长]　带步长的切片，步长有方向。
    
    ```py
    c=[1,2,3,4,5]
    c[4:0:-1]     # [5,4,3,2]
    c[4: : -1]    # [5,4,3,2,1]
    c[-2: : -2]   #[4,2]
    ```

+ 修改：　列表名[索引号]　＝　新值
+ 删除：　del 列表名[索引号]
+ 插入：　列表名.insert(出入位置索引号，新元素)

3. 字典｛｝

+ 字典放着{建：值，建：值}

    ```py
    dic = {1: '123',"name":"zhangsan"}
    ```
+ 用字典名[键]索引字典中的值
    ```py
    dic["name"] #表示字典dic中建"name"对应的值"zhangsan"
    ```
+ 修改：字典名[建] = 新值
+ 删除：del字典名[建]
+ 插入：字典名[新键] = 新值
